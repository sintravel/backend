# 신트레블 Backend REST API 

## 프로젝트 소개 

여행지 정보를 한 곳에 모아 사용자들에게 여행지 정보를 제공해주고, 사용자 성향 기반의 여행지 추천 및 커뮤니티 앱 개발 

## 사용한 기술
Java 11
Spring boot 
Spring Data JPA 
Spring Security
Spring batch 
Mysql
